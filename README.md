# 🚀 FocusFlow — Stay in Flow, Build Discipline

> A modern SwiftUI productivity app designed to help users eliminate distractions, build consistency, and achieve deep focus through structured daily tracking.

---

## 📱 Overview

**FocusFlow** is a clean and modern iOS productivity app built using **SwiftUI**.  
It helps users stay focused, track progress, maintain streaks, and build better daily habits through a minimal and distraction-free interface.

The project focuses on:
- Real-world productivity system design
- Smooth SwiftUI animations
- Clean UI/UX inspired by modern iOS apps

---

## ✨ Features

- ⏱ Focus Timer for deep work sessions  
- 📊 Productivity analytics dashboard  
- 🔥 Daily streak tracking system  
- 🏆 Achievement system  
- 👤 Modern profile section  
- 🎨 Glassmorphism UI design  
- ⚡ Smooth SwiftUI animations  
- 📱 Fully native iOS experience
